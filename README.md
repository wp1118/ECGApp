# 心电图标准波形绘制 App

**版权所有 © 2025 woP All Rights Reserved**

## 应用简介

这是一款用于学习和练习绘制标准心电图波形的安卓应用。用户可以按照标准的心电图波形顺序（P波→PR间期→QRS波群→ST段→T波）进行绘制，系统会实时检测绘制的波形是否符合医学标准。

**核心特色**：支持振幅限制检测（QRS>0.5mv、Q<1/4R、T>R/10）和ST段形态分析。

**开发者：woP**

## 主要功能

### 1. 心电图坐标纸背景
- 粉色小格：每格代表 0.04秒（横轴）/ 0.1mv（纵轴）
- 深粉色大格：每5小格为1大格，代表 0.2秒 / 0.5mv

### 2. 分阶段绘制
按照标准心电图波形顺序，依次绘制：
1. **P波** - 向上的圆钝波形
2. **PR间期** - 基线上的水平段
3. **QRS波群** - Q(下)→R(上尖)→S(下)
4. **ST段** - 回到基线，检测弓背上抬
5. **T波** - 向上的圆钝波形

### 3. 实时检测与提示
每完成一个阶段的绘制，系统会自动检测：
- 时间参数是否在正常范围内
- 幅度参数是否超标
- **QRS振幅是否>0.5mv**
- **振幅比例是否符合标准（Q<1/4R, T>R/10）**
- **ST段是否为弓背上抬（异常形态）**

**如果不符合标准，会弹出"不正常"警告，并说明具体原因。**

## 心电图标准参数

### P波
| 参数 | 标准值 |
|-----|-------|
| 时间 | ＜0.12秒（3小格） |
| 幅度 | ＜0.25mv（2.5小格） |
| 形态 | 向上圆钝 |

### PR间期
| 参数 | 标准值 |
|-----|-------|
| 时间 | 0.12-0.20秒（3-5小格） |
| 位置 | 在基线上 |

### QRS波群 ⭐新增振幅限制
| 参数 | 标准值 |
|-----|-------|
| Q波时间 | ＜0.04秒（1小格） |
| Q波幅度 | ＜1/4 R波 |
| **R波振幅** | **＞0.5mv（5小格）** |
| QRS总时间 | ＜0.12秒（3小格） |
| 形态 | Q(小降)→R(尖升)→S(小降) |

### ST段 ⭐新增形态检测
| 参数 | 标准值 |
|-----|-------|
| 压低 | ＜0.05mv（0.5小格） |
| 抬高 | ＜0.1mv（1小格） |
| **形态** | **警惕弓背上抬！（异常）** |
| 正常形态 | 水平或轻微下斜 |

### T波 ⭐新增比例检测
| 参数 | 标准值 |
|-----|-------|
| 幅度上限 | ＜0.5mv（5小格） |
| **幅度下限** | **＞R/10（R波的1/10）** |
| 方向 | I、II、V5-6导联直立 |
| 形态 | 向上圆钝 |

## 振幅检测说明

### QRS振幅检测
- **检测时机**：QRS波群绘制完成后
- **判定标准**：R波振幅必须大于0.5mv
- **临床意义**：QRS振幅过低可能提示低电压或心包积液

### Q/R比例检测
- **检测时机**：QRS波群绘制完成后
- **判定标准**：Q波幅度必须小于R波的1/4
- **临床意义**：Q波过深可能提示陈旧性心肌梗死

### T/R比例检测
- **检测时机**：T波绘制完成后
- **判定标准**：T波幅度必须大于R波的1/10
- **临床意义**：T波过低可能提示心肌缺血

## 使用方法

1. **启动应用**：打开App，看到心电图坐标纸背景和当前阶段提示
2. **开始绘制**：用手指在屏幕上按顺序绘制波形
3. **完成阶段**：松开手指后，系统会自动检测
4. **继续下一波**：检测通过后会自动进入下一阶段
5. **重新绘制**：如果不正常，可以点击"重新绘制"按钮

### 绘制技巧
- **P波**：轻轻向上画一个圆钝的"小山坡"
- **PR间期**：在基线上水平向右画直线
- **QRS波群**：
  - Q波：轻微向下
  - **R波：快速向上画尖峰（必须高于5小格！）**
  - S波：再轻微向下回到基线
- **ST段**：在基线上轻微水平延伸
- **T波**：向上画比P波稍大的圆钝波形

## 操作提示

- 每个阶段都有文字提示，说明当前需要绘制的波形特点
- 点击"标准说明"按钮可以查看详细的心电图标准
- 绘制时请尽量平滑，从上一阶段的终点继续
- **注意：QRS波群的R波必须高于5小格（0.5mv）！**

## 编译安装

### 使用 Android Studio
1. 打开 Android Studio
2. 选择 "Open an existing Android Studio project"
3. 选择 ECGApp 文件夹
4. 等待Gradle同步完成
5. 点击 Run 按钮运行

### 使用命令行
```bash
cd ECGApp
./gradlew assembleDebug
```
生成的APK文件位于：`app/build/outputs/apk/debug/app-debug.apk`
